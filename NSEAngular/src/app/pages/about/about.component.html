📖 About NSE Data Dashboard

About This Application

The NSE Data Dashboard is a single-page application built with Angular that displays market data from the National Stock Exchange (NSE) of India. The UI consumes a Django REST Framework API, which fetches and stores data from NSE's public allIndices endpoint.

Features

  • 139+ Indices: NIFTY 50, NIFTY BANK, NIFTY MIDCAP, and many more
  • Live Data: Current prices, changes, and percentage changes
  • Detailed Metrics: OHLC data, P/E ratios, P/B ratios, dividend yields
  • Performance Tracking: 30-day and 365-day performance data
  • Historical Snapshots: Price history recorded on every refresh
  • Search & Filter: Instant client-side search and category filters
  • Responsive Design: Works on desktop and mobile devices

Technical Stack

Frontend: Angular 19 (standalone components, signals)
Backend: Django 5 + Django REST Framework
Database: SQLite (indices + history snapshots)
Data Source: NSE allIndices API

API Endpoints (Django)

Endpoint Description Returns
GET /api/indices/ Paginated list of all indices JSON
GET /api/indices/<symbol>/ Detailed data for a specific index JSON
GET /api/indices/<symbol>/history/ Historical snapshots for an index JSON
GET /api/indices/refresh/ Pull fresh data from NSE into the database JSON
GET /api/status/status/ Overall API status and endpoint availability JSON
GET /api/status/health/ Simple health check JSON

Important: API Limitations

Individual Equity Stock Quotes

Status: Not Available

NSE has blocked direct API access to individual equity stock quotes. The endpoint https://www.nseindia.com/api/quote-equity?symbol=SYMBOL returns HTTP 403 (Access Denied). This dashboard therefore focuses on index-level data.

Available Data

NSE allIndices Endpoint - WORKING ✅

The primary data source is NSE's official /api/allIndices endpoint, providing price data, OHLC, 52-week highs/lows, valuation metrics, and market activity for 139+ indices.

How to Use

  1. Browse Indices: The home page lists every index in a searchable table
  2. Search: Use the search box to filter indices by name
  3. Filter by Category: Toggle Derivatives / Broad Market filters
  4. View Details: Click "Details →" on any index for full metrics and history
  5. Refresh: Use "Refresh from NSE" to pull the latest data into the backend
