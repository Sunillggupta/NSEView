📖 About NSE Data Dashboard

About This Application

The NSE Data Dashboard is a Python web application built with Flask that displays real-time market data from the National Stock Exchange (NSE) of India. It provides comprehensive indices information and market metrics.

Features

  • 139+ Indices: Access data for NIFTY 50, NIFTY BANK, NIFTY MIDCAP, and many more
  • Real-time Data: Current prices, changes, and percentage changes
  • Detailed Metrics: OHLC data, P/E ratios, P/B ratios, dividend yields
  • Performance Tracking: Daily, monthly, and yearly performance data
  • Market Activity: Advances, declines, and unchanged counts
  • Responsive Design: Works on desktop and mobile devices
  • Search & Filter: Find indices quickly with search and category filters

Technical Stack

Backend: Python, Flask
Frontend: HTML5, CSS3, JavaScript
Data Source: NSE allIndices API
HTTP Client: Requests library

API Endpoints

Endpoint Description Returns
GET / All indices in table format HTML
GET /index/<name> Detailed view for specific index HTML
GET /equity/<symbol> Equity quote (returns API limitation info) HTML
GET /api/indices All indices data JSON
GET /api/index/<name> Specific index data JSON
GET /api/status API status and available endpoints JSON

Important: API Limitations

Individual Equity Stock Quotes

Status: Not Available

NSE has blocked direct API access to individual equity stock quotes. The endpoint https://www.nseindia.com/api/quote-equity?symbol=SYMBOL returns HTTP 403 (Access Denied).

This application handles this limitation gracefully by:

  • Detecting the 403 error response
  • Returning informative status messages
  • Recommending alternative data sources
  • Providing index data as an alternative

Available Data

NSE allIndices Endpoint - WORKING ✅

The primary data source is NSE's official /api/allIndices endpoint, which provides:

  • Price data for 139+ indices
  • Day's OHLC (Open, High, Low, Close)
  • 52-week High and Low
  • Valuation metrics (P/E, P/B ratios)
  • Market activity (advances, declines)
  • Historical performance data

Alternatives for Individual Stock Data

If you need individual equity stock quotes:

  1. Third-Party APIs
    • Alpha Vantage - Free tier with limited requests
    • Finnhub - Good coverage of Indian stocks
    • IEX Cloud - Comprehensive data
  2. Official NSE Resources
    • NSE official API (available through proper channels)
    • NSE historical data downloads
  3. Web Scraping
    • Use Selenium for JavaScript-rendered pages
    • Implement polite request patterns
    • Respect Terms of Service
  4. Local Caching
    • Maintain local database of stock data
    • Update periodically from reliable sources
    • Serve cached data when APIs are unavailable

How to Use

  1. Browse Indices: Go to the home page to see all available indices in a sortable table
  2. Search Indices: Use the search box to find specific indices by name
  3. Filter by Category: Use filter buttons to show specific types of indices
  4. View Details: Click "Details" on any index to see comprehensive information
  5. API Access: Use JSON endpoints for programmatic access
