Why is this happening?

NSE (National Stock Exchange) has implemented strict API access controls and blocked the direct endpoint for fetching individual equity stock quotes. This was previously available at:

https://www.nseindia.com/api/quote-equity?symbol=SYMBOL

However, the allIndices endpoint is still available and provides comprehensive market data for indices and broader market information.

✅ What Still Works

  • Index Data: NIFTY 50, NIFTY BANK, NIFTY MIDCAP, and 130+ other indices
  • Market Metrics: P/E ratios, P/B ratios, dividend yields
  • Price Information: Open, High, Low, Close prices for indices
  • Performance Data: Daily, monthly, and yearly performance metrics

📚 Alternative Solutions

To get individual stock data, you can:

  1. Use NSE Index Data: The indices contain thousands of stocks. Use our available index data to track market trends.
  2. Third-Party APIs:
    • Alpha Vantage - Free tier available
    • Finnhub - Free tier for Indian stocks
    • IEX Cloud - Comprehensive market data
    • Official NSE API - Through proper channels
  3. Web Scraping: Use Selenium or Playwright to extract data from NSE's web pages (with proper delays to avoid server overload)
  4. Local Database: Maintain a cached database of frequently used stocks with periodic updates
